# API para Gerenciamento de pedidos 

Construção de API que permite gerenciar pedidos

# Pré-Requisitos:
- .Net 8
- Visual Studio

# Utilização de Técnicas e Tecnologias:
- Entity Framework InMemory
- DDD (domain-driven design
- Testes unitários(NUnit)
- swagger
